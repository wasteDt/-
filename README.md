# Node.js + socket.io 实时聊天室

一个不持久化聊天历史的多用户实时聊天室示例。服务端使用 `socket.io` 维护长连接，校验用户名唯一性，并把用户列表、系统提示和聊天消息实时广播给所有在线客户端。

## 运行

```bash
pnpm install
pnpm start
```

浏览器打开 `http://localhost:3000`，可以开多个窗口模拟多用户在线。

## 功能

- 多用户同时在线聊天
- 消息实时同步到所有客户端
- 用户名唯一性校验
- 用户名可进入后继续修改
- 在线用户列表实时更新
- 会话数据只保存在内存中，服务器重启后不会保留聊天历史
